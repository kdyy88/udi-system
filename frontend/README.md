# Frontend (Next.js 16 + shadcn/ui)

## 阶段 1 完成项

- Next.js 16 项目初始化
- shadcn/ui + `lucide-react`
- `axios` API 客户端
- 与后端 `/api/v1/health` 联调页面
- App Router 目录采用 `frontend/app`

## 安装与启动

```bash
pnpm install
pnpm dev
```

默认地址：`http://localhost:3000`

## 环境变量（可选）

```bash
NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8000
```
